üåü bat -- The Modern cat Replacement Link to heading

The default cat command works, but it lacks modern usability features. bat makes file viewing more powerful and readable with syntax highlighting, line numbers, and Git integration.

üîπ Why bat? Link to heading

‚úÖ Syntax highlighting – Works for code, configs, logs
‚úÖ Git integration – Shows diffs and untracked changes
‚úÖ Built-in paging – Scroll through large files automatically
‚úÖ Supports multiple files – Compare files side by side
‚úÖ Drop-in replacement for cat – Works the same way, just better

‚ö° Quick Start Link to heading

üì• Install: Link to heading

  • Ubuntu/Debian: sudo apt install bat

  • MacOS: brew install bat

üí° Common Usage: Link to heading

# Disable extra formatting for raw output

# Compare multiple files side by side

# Use bat in pipelines

üî• Why switch? bat makes reading files visually appealing and more functional while keeping cat’s simplicity.
